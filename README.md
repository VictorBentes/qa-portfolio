# 🧪 QA Portfolio - Victor Bentes

Este repositório contém meus estudos práticos na área de Quality Assurance (QA), com foco em testes manuais, criação de casos de teste e identificação de bugs em aplicações web.

## 📚 Conteúdo

- Casos de teste: cenários criados para validar funcionalidades
- Bugs encontrados: registro de falhas identificadas durante os testes
- Evidências: imagens que demonstram os testes realizados

## 🎯 Objetivo

Desenvolver minhas habilidades em QA, aplicando conceitos de teste na prática e evoluindo continuamente na área de qualidade de software.

## 🚀 Sobre mim

Atualmente estou em fase de aprendizado na área de QA, buscando minha primeira oportunidade profissional e me dedicando a evoluir minhas habilidades em testes e qualidade de software.
