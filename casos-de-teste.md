# Casos de Teste
## Teste de Login com dados válidos
**Objetivo:** Verificar se o usuário consegue logar com credenciais corretas
**Passos:**
1. Acessar a página de login
2. Inserir email válido
3. Inserir senha correta
4. Clicar em "Entrar"
**Resultado esperado:** Usuário deve acessar o sistema com sucesso
## Teste de Login com senha inválida
**Objetivo:** Verificar comportamento com senha incorreta
**Passos:**
1. Acessar a página de login
2. Inserir email válido
3. Inserir senha incorreta
4. Clicar em "Entrar"
**Resultado esperado:** Sistema deve exibir mensagem de erro
